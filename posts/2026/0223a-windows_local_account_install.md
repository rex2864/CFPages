## Windows local account install

win11 설치 후 최초 설정 진행 중에

"Microsoft 환경 잠금 해제" 화면이 나오면

Shift + F10을 입력하여 cmd 화면을 띄운다.

cmd 화면에 `start ms-cxh:localonly`를 입력한다.

그러면 로컬 계정 생성 화면이 뜬다.

ID/PW를 생성하고 다음으로 넘어가면 된다.

---

작업 관리자를 열어서 cmd를 실행한다음 아래 명령으로 bypass 시키면 된다.
